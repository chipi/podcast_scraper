GI / KG local viz

Prototype for issue #445: load .gi.json / .kg.json from disk, no backend. Two graph engines so you can compare; JSON-only mode skips graph libs.

  • Graph — vis-network Physics, filters (node types + hide ungrounded insights), pan/zoom/drag, fit + re-stabilize, Chart.js bars, CLI cheatsheet.
  • Graph — Cytoscape.js Same features with cose layout — pick a winner vs vis-network.
  • JSON + metrics + chart Overview, node-type pills, Chart.js distribution, raw JSON. Minimal deps.

From repo root: make serve-gi-kg-viz then open http://127.0.0.1:8765/ (avoids browser restrictions on CDNs under file://).

See web/gi-kg-viz/README.md for details.
