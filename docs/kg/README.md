# Knowledge Graph (KG) — documentation

This folder holds **Knowledge Graph Layer (KG)** artifacts referenced by [PRD-019](../prd/PRD-019-knowledge-graph-layer.md) and [RFC-055](../rfc/RFC-055-knowledge-graph-layer-core.md).

**Note:** **KG** is a **separate feature** from **Grounded Insight Layer (GIL / GI)**. See PRD-017 / RFC-049 for `gi.json` and grounded insights.
