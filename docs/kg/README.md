# Knowledge Graph (KG) — documentation

This folder holds **Knowledge Graph Layer (KG)** artifacts referenced by [PRD-019](../prd/PRD-019-knowledge-graph-layer.md), [RFC-055](../rfc/RFC-055-knowledge-graph-layer-core.md), and [RFC-056](../rfc/RFC-056-knowledge-graph-layer-use-cases.md).

**User guide:** [Knowledge Graph Guide](../guides/KNOWLEDGE_GRAPH_GUIDE.md).

**Note:** **KG** is a **separate feature** from **Grounded Insight Layer (GIL / GI)**. See PRD-017 / RFC-049 for `gi.json` and grounded insights.
